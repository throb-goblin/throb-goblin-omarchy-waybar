# throb-goblin's omarchy waybar
Simple and clean hyprland waybar config/ style. 

Music player pill disapears upon no music/ media playing. 

Includes on screen keyboard button (must be seperately configured. I configured wvkbd "https://github.com/jjsullivan5196/wvkbd")

To install add config.jsonc and style.css to ".config/waybar/" in Omarchy (Arch - Hyprland).
